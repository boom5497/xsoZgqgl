# 项目简介

## 核心功能
- 系统首页
- 采购员管理
- 销售员管理
- 供应商管理
- 商品库存管理
- 采购申请管理
- 进货入库管理
- 销售出库管理
- 商品报损管理
- 系统管理
- 个人中心

## 适用场景
- 中小企业商品进销存管理
- 数据可视化展示

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
